This guide will walk you through installing Lancer.

## Overview

The installation process takes about 5-10 minutes and will:

1. Check prerequisites (Node.js, Pulumi)
2. Collect your cloud provider credentials
3. Create your admin account
4. Deploy the infrastructure
5. Provide you with a dashboard URL
