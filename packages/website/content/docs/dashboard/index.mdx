The Lancer dashboard is a web interface for managing your deployments.

## Accessing the Dashboard

After installation, access your dashboard at the URL provided by the installer.

## Login

Use the admin credentials you created during installation:

1. Enter your **email**
2. Enter your **password**
3. Click **Login**

Your session will remain active for 7 days.

## Features

From the dashboard, you can:

- **Manage Providers**: Add and configure cloud provider credentials
